# Розробка вебсайту для мережі пекарень "Сімейна пекарня" з використанням сучасних технологій фронтенд-розробки.
Кваліфікаційна робота на підтвердження ступеня фахового молодшого бакалавра (ВСП «ППФК НТУ «ХПІ»). Керівник роботи – Барабан Дмитро Миколайович.

## Завдання до дипломної роботи

Продукт має являти собою односторінковий вебзастосунок з високим рівнем продуктивності та унікальним дизайном. Семантична верстка HTML5, стилізація за допомогою CSS3 (або препроцесорів Sass/Less), мова програмування JavaScript (можливе використання бібліотек/фреймворків). Сайт повинен містити інтерактивні елементи (галереї, форми, анімації), бути повністю адаптивним та оптимізованим для швидкого завантаження. За основу взяти структуру та контент сайту https://family-bakery.com.ua/

## Використані технології:
* HTML5
* CSS3
* Sass / Less
* JavaScript
